# Event Booking System

## Описание
Уеб приложение, разработено с Django, за управление и резервиране на събития.

## Функционалности
- Регистрация и вход
- Създаване на събития
- Редактиране на събития
- Резервиране на събития
- Категории
- Обратна връзка

## Използвани технологии
- Python
- Django
- PostgreSQL
- Pillow

## Инсталация

pip install -r requirements

python manage.py migrate

python manage.py runserver
